  ### study
  1、UG NX 10.0 运动仿真与分析教程，已学习内容：连杆、运动副、传动副、连接器；

  4、ROS是一种开源的机器人系统；

  1、复杂的函数和高数会影响进度；  
  2、缺少机械原理和机械结构相关的知识；  
  3、了解的越多，越能感觉到机器人设计的复杂，无论是结构系统还是控制系统，  
   ### plan  
  1、了解市场，确定切入点，什么类型、什么材料、什么结构、怎么控制等等；  
  2、补充机械原理和结构相关知识；  
  3、暂时不考虑过于复杂高深的内容，先打好基础；
